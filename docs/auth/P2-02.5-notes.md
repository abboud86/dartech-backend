# P2-02.5 — Housekeeping/Harden (Auth)
- Erreurs JSON uniformes (error/detail), Content-Type explicite.
- CORS: OPTIONS public, jamais 401/429.
- Rate limiters ciblés refresh/logout (prod/dev), désactivés/relax en test.
- TokenAuthenticator: check revoked/expired + update lastUsedAt.
- Test authenticator (X_TEST_USER) actif seulement en env test.
- Qualité+Tests: cs/phpstan/phpunit doivent rester verts.

Cette note n'introduit aucun changement runtime; elle sert de guide d'implémentation.
